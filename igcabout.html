About IGC Webview

© 2015 Alistair Malcolm Green and Richard Brisbourne

IGCWebview is an open source program written almost entirely in Javascript, although it makes use of some server-side php code to interrogate databases. Copyright is held jointly by Alistair M Green and Richard Brisbourne; for further information for how to access the source code see below, for information on how you are entitled to make use of it see the licence .

Credits- program code
The Javascript code for the application itself was mostly written by Alistair Green with some embellishment by Richard Brisbourne. However, like most open source projects IGC Webview makes extensive use of third party libraries available under similar open source licences:

  • Leaflet.js- an interactive map library
  • JQuery- a javascript library well-know to web developers
  • Flot charts- a JQuery add-in for chart plotting
  • Flot Axis Labels plugin- for a full list of contributors see the link
  • Leaflet-semicircle- Leaflet plug-in by Jan Pieter Waagmeester ('Jieter') used for sector drawing

Credits- icons
Use is made of icons from Font Awsome

Credits- data
Data is drawn from a number of sources- some are license-free, some under the Open Database Contents licence, some under individual licenses. For further details follow the links:

Of prime importance is mapping- maps are sourced from Open Street Map.

Timezone data is sourced from the timezonedb web service

We use copies of waypoint files from the British Gliding Association and the Welt 2000 database.

Airspace data is drawn from a number of places, the largest being soaringweb.org. Unfortunately very little is available in computer-readable format from official sources so we are dependant on people who work voluntarily to create the files we all use in our software, both on the ground and in our gliders, and who deserve more credit than they often get. The data in IGCWebview is actually held in tables on the server, so has been arrived at by conversion of some of these files. In some cases copyright has been asserted on the files, in those cases there are links below to the originals containing copyright statements.
